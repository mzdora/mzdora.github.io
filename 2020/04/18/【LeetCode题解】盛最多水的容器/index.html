【LeetCode题解】盛最多水的容器

今天的每日一题是一道双指针的问题。

显而易见，双指针就是一个指针指向数组/字符串的最左边那个元素，另一个指针指向最右边的元素。然后经过一系列的条件进行判断，得出结果。

题目链接：https://leetcode-cn.com/problems/container-with-most-water/

题目描叙：

给你 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

题目分析：

我们看到这种数组题，第一反应应该就是双指针了。如果你现在脑袋里反应不过来需要用双指针的话，证明你刷题刷得还不够多。我们首先把左指针放在数组的 height[0] 位置，然后把右指针放在数组的 height[height.length -1] 的位置(保证在边界内)。

接着我们考虑笛卡尔坐标系的x轴，因为我们的双指针技巧是对左右指针不断维护的。所以我们的x轴的长度也需要不断维护。最开始的时候，我们的长度是 len = height.length -1 。然后每一次维护呢就对其 len -= 1 。

明白上面的描叙之后，那么我们的最大面积理所当然的也通过这样的循环不断地被维护。
