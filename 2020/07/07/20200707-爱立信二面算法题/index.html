1.求最小位数

题目描叙：

Write a function solution that given an integer N,returns the smallest number with the same number of digits. You can assume N is between 1 and 10^9

For example,given N=125,the function should return 100.Given N = 10,the function should return 10.Given N=1,the function should return 0.

题目分析:

水题，没什么好分析的。因为题目数字是到10^9,O(2*n)的算法复杂度完全可以胜任。首先while循环求出位数个数，然后再乘就可以了。

2.求类A..AB..B式字符串最小删除值

题目描叙:

We are given a string S of length N consisting only of letters ‘A’ and/or ‘B’.Out goal is to obtain a string in the format “A..AB..B”(all ‘A’ occur before all ‘B’) by deleting some letters from S. In particular,strings consisting only of ‘A’ or only of ‘B’ fit this format.

Write a function:

That,given a string S,returns the minimum number of letters that needs to be deleted from S in order to obtain a string in the above format.

Example:

S=”BAAABAB” return 2

S=”BBABAA” return 3

S=”AABBBB” return 0

题目分析:

任意确定一个A和B的边界，然后删除边界左边的所有B和边界右边的所有A。最后进行取最大值就可以了。

4.钉钉子

题目描叙:

There are N nails hammered into the same block of wood.Each nail sticks out of the wood at some length.You can choose at most K nails and hammer them down to any length between their original lengths and 0.Nails cannot be pulled up.The goal is to have as many nails of the same length as possible.

You are given an implementation of a function:

which,given an array A of N integers representing lengths of the nails sorted in a non-decreasing oreder and an integer K,returns the maximal number of nails that can be positioned at the same length after hammering down at most K nails

For example,given K=2 and array A = [1,1,3,3,3,4,5,5,5,5] the function should return 5.One of the possibilities is to hammer the nails represented by A[8] and A[9] down to length 3.

题目分析:

贪心算法，因为数组已排序。直接把最后K个元素抹掉。然后找出best长度之后加上就好。
