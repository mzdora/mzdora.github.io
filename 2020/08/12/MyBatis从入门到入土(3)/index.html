3.MyBatis中的缓存设计

一级缓存默认开启，缓存范围SqlSession会话

二级缓存手动开启，属于范围Mapper Namespace

二级缓存运行原则:

  1. 二级缓存开启后默认所有查询操作均使用缓存
  2. 写操作commit提交时对该namespace缓存强制清空
  3. 配置useCache=false可以不用缓存
  4. 配置flushCache=true代表强制清空缓存

如何在Mapper.xml中配置二级缓存？

这样就是给namespace申明开启二级缓存。

eviction 表示缓存所使用的算法,这里使用的LRU(最少使用)算法

  • LRU 最少使用算法
  • FIFO 先进先出算法
  • SOFT 软引用(没屁用)
  • WEAK 弱引用(没屁用)

flushInterval 是缓存清空间隔

size 是缓存的存储容量

readOnly 是缓存是否只读
